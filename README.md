# AWS ECS Fargate

## Introduction 

A PaaS like workflow for AWS ECS Fargate Infastructure as Code, powered by CTO.ai

## Pre-requisites

- Docker, Node (NVM) 12+ & npm installed
- Sign up for CTO.ai, setup CTO.ai team
- Install Ops CLI, Connect Github & Slack
